标识符
C 标识符是用来标识变量、函数，或任何其他用户自定义项目的名称。

一个标识符==以字母 A-Z 或 a-z 或下划线 \_ 开始==，后跟零个或多个字母、下划线和数字（0-9）。

C 标识符内==不允许出现标点字符==，比如 @、\$ 和 %。C 是==区分大小写==的编程语言。
关键字
关键字==不能==作为常量名、变量名或其他标识符名称
