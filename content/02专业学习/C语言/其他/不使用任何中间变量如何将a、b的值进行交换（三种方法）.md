通过相加减可以达到两数交换的目的，缺点就是有可能会产生数据溢出的情况
使用乘除使两数交换，缺点也是可能会溢出
对两数进行异或运算，利用异或的自反性，这种方法效率最高，且不会产生越界。**（推荐使用）**

```c
int a = 5, b = 15;
a = a ^ b;
b = a ^ b;
a = a ^ b;
```
